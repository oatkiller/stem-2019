Basics Write pairs of numbers in the textarea. Each pair forms an point. Odd numbers represent 'x's and even numbers represent 'y's. All points are connected in order. The last point connects to the first one.
Stroke? Use the stroke checkbox to determine if the path should be filled in or drawn with a stroke.
Style Each path can have a style. Use the color input to select a color for each path.
Paths The 'clone' button adds a second path. Paths aren't connected. Additional paths can be removed using the 'remove' button.
